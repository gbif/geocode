GBIF Reverse Geocoder Debugger

The debug map shows the geospatial layer polygons used by GBIF’s occurrence processing. Click the map to query the reverse geocoder. The result is the ISO code (e.g. ‘DK’), not the name.

Data sources are documented in the source code.

This is a debugging interface primarily for internal use at GBIF. The names of countries, islands and territories are from the original data sources, and do not necessarily reflect GBIF policy.
